cd -> change directory 
cd .. -> vamos una carpeta hacia atras

cd nombreCarpeta -> viaja hacia la carpeta

ls -> nos muestra el contenido

git status -> nos muestra las modificaciones

git add -> nos agrega los archivos

origin: https://github.com/benvenutim93/ClaseGitHub.git

git commit -m "Comentario entre comillas"

git push -> envia la informacion al repositorio externo

git push origin(url) master(rama principal)

git pull origin master -> se obtiene el repositorio actualizado desde github
